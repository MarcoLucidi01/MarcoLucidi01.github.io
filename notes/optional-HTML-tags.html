optional HTML tags

I stumbled across Google's HTML/CSS Style Guide the other day on the train, on my way back home and started reading.

I learned that lots of tags can be omitted (<html>, <head>, </p>, </li>, ...) because they are implicit. They still will be in the DOM, but it's not necessary to write them in the source.

The HTML spec explains it better.

Imho this is really good news since it reduces file size, boilerplate and increases readability of sources.
