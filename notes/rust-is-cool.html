rust is cool

started learning rust recently for a side project. I'm at chapter 8 of The Book. I'm really a newbie and I have a lot to grasp yet, but the experience so far as been positive!

some random points about rust:

  • everything is immutable by default
  • OOP is supported but NOT IMPOSED (fuck you java)
  • there are no wrapper for generics (fuck you java)
  • there are no exceptions (fuck you java)
  • there is the match keyword which is awesome. It's similar to a switch but it's much much more powerful. It's a switch on steroids! Look!
  • there is no NULL! The concept of NULL is expressed with the enum variant Option::None. Option enum can hold either Some(T) value or None. It's like Optional in java, but more convenient to use thanks to match.
  • the compiler is a bit slow, but gives you excellent error messages!
  • this whole idea of RAII, default immutability and memory checks at compile time seems a really gooood one! It lets rust to run blazingly fast (like C), to not have a GC nor a runtime and to provide good abstractions at near to 0 cost!
