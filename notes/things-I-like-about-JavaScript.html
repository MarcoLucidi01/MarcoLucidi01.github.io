things I like about JavaScript

functions

Functions in JavaScript can be stored in variables, objects and arrays. Functions can be passed as arguments to functions, and functions can be returned from functions.

They can be defined inside other functions and get access to parameters and variables of the functions they are defined within. This behavior is called closure and it's soooo useful if used properly.

Functions are first class objects and I love this thing. They allow you to write very expressive code which is one of my main goal as a programmer.

objects

Objects in JavaScript are containers (collections) of key-value properties. A key can be any string, a value any JavaScript value. They can contain other objects so they can easily represent e.g. a tree or a graph.

They can be easily initialized through object literals (which inspired the JavaScript Object Notation aka JSON format) and easily updated.

They provide a very powerful and flexible associative array like data structure built-in the language.
